# Методи тестування

Проект: *"NotatkaHUB"*

Команда: *"СВАРЩИКИ"*

## Перелік методів тестування 

| № |     Тип тестування        |     Оригінальна назва     |                                          Опис методу                                             | 
|---|---------------------------|---------------------------|--------------------------------------------------------------------------------------------------|
| 1 | Компонентне               | Unit Testing              | Єдиний вид тестів для розробника. Перевірка логіки збереження тексту, валідації пароля тощо      |
| 2 | Функціональне             | Functional Testing        | Перевірка сценаріїв: «створити нотатку», «видалити», «пошук за словом».                          |
| 3 | Конфігураційне            | Compatibility Testing     | Перевірка роботи сайту в різних браузерах (Chrome, Safari) та на мобільних ОС.                   |
| 4 | Продуктивності            | Performance Testing       | Вимірювання часу завантаження головної сторінки та швидкості пошуку в базі даних.                |
| 5 | Стрес-тестування          | Load Testing              | Імітація великої кількості користувачів, що одночасно створюють нотатки                          |
| 6 | Безпеки                   | Security Testing          | Перевірка надійності авторизації та захисту від ін'єкцій у поля вводу.                           |
| 7 | Регресійне                | Regression Testing        | Повторна перевірка функцій після кожного виправлення багів, щоб переконатися, що все інше працює.|

## Інструменти тестування

| № |         Назва           |          Вид тестування          |                       Сайт програми                              |
|---|-------------------------|----------------------------------|------------------------------------------------------------------|
| 1 | **jUnit / NUnit**       | Юніт-тестування (Unit)           | [junit.org](https://junit.org/) /[nunit.org](https://nunit.org/) |
| 2 | **Telerik Test Studio** | Функціональне (Functional)       | [telerik.com](https://www.telerik.com/teststudio)                |
| 3 | **BrowserStack**        | Тести сумісності (Compatibility) | [browserstack.com](https://www.browserstack.com/)                |
| 4 | **Google Lighthouse**   | Тести швидкодії (Performance)    | [web.dev](https://developers.google.com/web/tools/lighthouse)    |
| 5 | **Fiddler**             | Тести швидкодії / Трафік         | [telerik.com/fiddler](https://www.telerik.com/fiddler)           |
| 6 | **Loadium**             | Навантажувальне (Load)           | [loadium.com](https://loadium.com/)                              |
| 7 | **Crashtest Security**  | Тестування безпеки (Security)    | [crashtest-security.com](https://crashtest-security.com/)        |
